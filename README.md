# VPN Ping Checker

Автоматический парсинг и проверка VPN ключей каждые 6 часов.

## Как это работает

1. Скрипт собирает ключи из указанных источников (обновляются каждые 2 дня)
2. Проверяет пинг до каждого сервера
3. Сохраняет только те ключи, у которых пинг < 70ms
4. Результат сохраняется в `FREE-VPN-FROM-KIRILL.txt`

## Автоматическое обновление

Файл обновляется каждые 6 часов через GitHub Actions.

## Использование

Просто вставьте в свой клиент ссылку https://free.vpnfromkirill.workers.dev/
